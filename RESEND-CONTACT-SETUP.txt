THERM-X CONTACT FORM — VERCEL + RESEND SETUP
=============================================

WHAT IS UPDATED
---------------
1. The contact form now sends to /api/contact.
2. api/contact.js is a secure Vercel Serverless Function.
3. The Resend API key stays on the server and is never exposed in browser code.
4. Visitor email is configured as Reply-To.
5. Up to 3 attachments are supported (2 MB each, 3 MB total).

FILES ADDED / UPDATED
---------------------
api/contact.js
assets/js/contact-resend.js
contact.html
.env.example

ADD THESE ENVIRONMENT VARIABLES IN VERCEL
-----------------------------------------
Open:
Vercel Dashboard > website project > Settings > Environment Variables

Add all 3 values:

RESEND_API_KEY
Value: re_your_real_resend_api_key

CONTACT_TO_EMAIL
Value: sales@thermxinnovations.com
(You may replace this with the inbox that must receive contact enquiries.)

CONTACT_FROM_EMAIL
Value: Therm-X Website <enquiries@thermxinnovations.com>
(The sender address must use your verified Resend domain.)

For each variable, enable Production, Preview, and Development if required.
Then redeploy the latest deployment.

HOW TO CREATE THE RESEND API KEY
--------------------------------
1. Open Resend Dashboard.
2. Open API Keys.
3. Click Create API Key.
4. Give it a name such as Therm-X Website.
5. Copy the key once.
6. Paste it only into the Vercel RESEND_API_KEY environment variable.

IMPORTANT SECURITY RULE
-----------------------
Do not paste the Resend API key directly into contact.html or contact-resend.js.
Frontend code is visible to everyone. The key belongs only in Vercel Environment Variables.
